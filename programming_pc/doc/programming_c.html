Learn to program in C and C++

After completing the above, you are ready to learn C. C is a very powerful programming language. With it, you can write games, or any other programs. You can even write your own operating system! Linux, for example is written in C. Once you can program C, other languages are very easy to learn. Start reading the Teach Yourself C book available from the library. You can try out the examples in the book on this computer. As you try writing your own programs, you may crash this computer. Don't worry, just press the triangle button at the bottom of the computer case to restart this computer.

To compile your program, use gcc, for example:
gcc hello.c -o hello
The above will compile the hello.c program and create an executable file called hello You can now run the executable file by typing:
./hello
Go ahead and try it from the command-line (meaning exit this program, then type the above)!

To edit your programs, you can use vi, or nano. Nano is easier, but you can also learn to use vi by reading Chapter 19 of the Using Linux book. For example, to edit the hello.c program, at the command-line, type:
nano hello.c
To exit the editor, press Ctrl and while holding it down, press x, then release both keys. This is shown as ^X at the bottom of the screen. Press Ctrl and g to get more help.
